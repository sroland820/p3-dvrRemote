PROJECT DEFINITION: Why does this project need to exist? This project needs to exist because peoples needs are changing and they are wanting to be able to do more with their remote. With advanced technology there should be software that accompanies it to make your life easier. What aspects of this project are good user experiences? Good user experience is going to be ease of use and fast response time. What aspects of this project are bad user experiences? Bad user experience would be confusing to use and slow delays. Describe the user(s) of, and also how they are using -- this project? The users is any TV watcher. They can be using this remote either through the device or their phone. What solutions may exist to improve the overall User Experience (UX)? Making a remote that is also attachable to your phone so you don't have to go running around looking for the remote.
